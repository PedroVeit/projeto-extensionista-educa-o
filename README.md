# projeto-extensionista-educa-o
Projeto de análise da influência de fatores socioeconômicos no desempenho dos estudantes do ENEM.
Alunos: Caihan Garcia, Pedro Passero Veit e Gustavo Silveira Goulart 


*Fase 1 - Compreensão do Negócio*

  1. Questão de pesquisa 
Como fatores socioeconômicos influenciam o desempenho dos estudantes brasileiro no ENEM? 
Buscaremos responder essa pergunta com dados através de análises de:
- Relação entre renda familiar e desempenho
- Influência do tipo de escola
- Escolaridade dos pais e desempenho
- Relação entre condições socioeconômicas e resultados educacionais

  2. Justificativa da relevância 
O ENEM é hoje a principal porta de entrada para o ensino superior no Brasil, sendo utilizado tanto por universidades públicas quanto por programas como o SISU, o ProUni e o FIES. Por isso, o desempenho no exame tem impacto direto sobre as oportunidades de acesso à educação. Estudar como fatores socioeconômicos influenciam esse desempenho é relevante porque permite identificar até que ponto as condições de origem do estudante ajudam a explicar as diferenças nos resultados. Além disso, o uso dos microdados do ENEM, que são públicos e abrangem milhões de participantes, oferece uma base concreta para analisar essas questões a partir de dados reais, contribuindo para um debate mais informado sobre a educação no Brasil. 

  3. Objetivos do projeto 
Temos como objetivo principal a compreensão de como os fatores socioeconômicos influenciam o desempenho dos estudantes no ENEM. Também temos como objetivo identificar a relação de renda com o desempenho do estudante, comparar o desempenho dos estudantes de escolas públicas e privadas, avaliar a escolaridade dos pais dos estudantes desenvolvendo dashboards para realizar as análises.

  4. Bases de dados selecionadas 
Selecionamos inicialmente a base pública de dados dos microdados do ENEM, disponibilizada pelo Instituto Nacional de Estudos e Pesquisas Educacionais 
Anísio Teixeira.
Base principal: Microdados do ENEM – INEP (https://www.gov.br/inep/ptbr/acesso-a-informacao/dados-abertos/microdados) 
A base contém informações referentes aos participantes do exame, como as notas obtidas nas áreas avaliadas, alguns dados socioeconômicos, informações sobre 
a renda familiar do estudante, escolaridade dos responsáveis, tipo da escola e algumas características demográficas e educacionais. Escolhemos essa base de dados por conta da disponibilidade pública dos dados, grande volume de informações relevantes para as análises, contendo variáveis socioeconômicas alinhadas com o objetivo de análise do projeto.
